[[..]] style wikilinks can link to not only Markdown files, but also to any other files.

For example, here is a link to some text file: File:Sample.txt

You can also specify the full, or subset of, the path: File:demo/Sample.txt

Of course, a custom link text may also be specified: Some text file

You can also add emails, but they are not considered as links: mailto: foo@bar.com
